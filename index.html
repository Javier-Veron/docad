Bienvenido a DOCAD

DOCAD es una aplicación web diseñada para facilitar la gestión administrativa de docentes en escuelas. Ofrecemos herramientas para la planificación de clases, seguimiento de asistencia, gestión de calificaciones y más.

Funcionalidades Principales

  • Planificación de clases
  • Seguimiento de asistencia
  • Gestión de calificaciones
  • Comunicación con estudiantes y padres
  • Informes y análisis de datos
